# 📊 Zepto Product Analysis using SQL

## 📌 Project Overview
This project analyzes Zepto's product catalog using MySQL. The analysis focuses on pricing, discounts, inventory management, stock availability, and category performance to generate business insights.

## 🛠️ Tools Used
- MySQL
- SQL
- GitHub

## 📂 Dataset
- Dataset: Zepto Product Dataset
- Total Products: 3000+
- Format: CSV

## 📈 SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- Aggregate Functions
- CASE Statements
- INNER JOIN
- Subqueries
- Window Functions (ROW_NUMBER)

## 📊 Business Questions Solved
- Total number of products
- Category-wise product analysis
- Average discount by category
- Highest inventory value
- Products with maximum MRP
- Top discount products
- Inventory contribution by category
- Out-of-stock analysis
- Price per gram analysis
- Top 3 products per category
- 25 SQL business questions in total

## 📌 Key Insights
- Categories with the highest inventory value require better inventory planning.
- Products with above-average discounts are suitable for promotional campaigns.
- High out-of-stock percentages may indicate strong customer demand or inventory shortages.
- Price-per-gram analysis helps identify products offering the best value.
